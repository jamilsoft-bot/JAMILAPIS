# Google Drive SDK Roadmap

* Add chunked/resumable uploads for large files.
* Add pagination helpers and iterators.
* Implement batch operations (bulk delete/update).
* Add permissions management (share/unshare, roles).
* Add move/copy helpers across folders/drives.
* Integrate Drive activity logs.
* Support push notifications (webhooks).
* Improve caching and rate-limit handling.
* Add CI test coverage and fixtures.
